## Instalação do Yarn (Caso você não tenha)
```
npm install -g yarn
```

## Instalação das dependências
```
yarn install
```

## Execução dos scripts
- Gerar documentação: `yarn doc`
- Rodar o projeto em modo de atualização contínua: `yarn dev`
- Buildar o projeto: `yarn build`
